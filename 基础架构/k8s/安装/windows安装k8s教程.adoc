== 安装 docker-desktop

=== 前提
* 主板中开启虚拟化
* 开启 windows 子系统

image::image/1.png[]

* 开启wsl2并安装 ubuntu 20.4 https://docs.microsoft.com/zh-cn/windows/wsl/install-win10[教程]



=== 下载镜像 

* 管理员 powershell 执行

```
 Set-ExecutionPolicy -ExecutionPolicy Unrestricted -Scope CurrentUser
```

* 确定版本

image::image/2.png[]

* 下载github代码并切换到对应版本下载镜像

=== 开启k8s 

image::image/3.png[]

```
大概 2-3分钟
```
